About the brand

FAQ

Quick answers

Is this a real store checkout?

Not yet. This is a polished front-end starter site. You can later connect Shopify, Snipcart, Stripe, or another cart system.

Can I host this on GitHub Pages?

Yes. Upload the files as-is to a GitHub repository and enable Pages. No build step is required.

Can the images and text be replaced?

Yes. Swap the assets in the assets folder and update the copy in index.html.
