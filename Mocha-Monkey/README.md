# Mocha Monkey Static Website

This is a simple static apparel brand website built for GitHub Pages or any basic web host.

## Files
- `index.html` — main page
- `styles.css` — styling
- `script.js` — theme toggle
- `assets/` — image assets used by the site

## How to host on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repo.
3. In GitHub, go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/root** folder.
6. Save.
7. GitHub will publish the site and give you a live URL.

## Notes
- This is a front-end only site.
- The "Add to cart" buttons are visual only for now.
- You can later connect a real cart system like Shopify Buy Buttons, Snipcart, or Stripe checkout.
